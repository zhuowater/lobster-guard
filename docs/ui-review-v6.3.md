# 🦞 龙虾卫士 Dashboard UI Review — v6.3.0

**评审人视角**: 资深企业安全运维负责人，评估是否值得向团队推广
**评审日期**: 2026-03-17
**评审维度**: 视觉设计 / 交互体验 / 信息架构 / 专业度 / 企业级特征

---

## 总体评分：6.5 / 10 — "能用，但不想给老板看"

功能基本完整，但视觉和交互停留在"开发者自己写的后台"水平，离 Grafana / DataDog / 阿里云安全中心这类产品差距明显。

---

## 🔴 严重问题（阻碍采购的）

### 1. 图标系统完全缺失
- 侧边栏导航使用方块占位符（□），看起来像 Unicode 渲染失败
- StatCard 的图标也是方块占位符
- **企业客户第一印象就毁了** — 看起来像"没做完的半成品"
- **修复**: 使用 SVG 内联图标（Lucide / Heroicons 风格），不需要引入图标库

### 2. 统计卡片数据异常
- 概览页 4 张卡片，最右侧被截断看不到
- 拦截数 / 告警数都是 0，但有请求趋势图 — 视觉上矛盾
- 卡片内图标和数字排列混乱，层次不清
- **修复**: 卡片响应式布局（grid auto-fit），图标+数字+标签三行垂直居中

### 3. 运行时间显示荒谬
- 顶栏显示 `22m21.534308958s` — 精确到纳秒？
- **企业客户看到这个只会觉得可笑** — 这是调试信息不是产品展示
- **修复**: 格式化为 `22 分钟` / `3 天 5 小时` / `uptime: 72h`

### 4. 空状态体验极差
- 规则命中率页 "暂无命中数据" — 只有一行灰字 + 一个方块
- 监控页 "暂无攻击事件" — 同样寒酸
- **修复**: 设计空状态插画（SVG），加引导文案（"规则正在保护中，命中后这里会显示统计"）

---

## 🟡 明显问题（影响使用体验的）

### 5. 颜色语义混乱
- 青色 (#00d4ff) 被过度使用：导航高亮、链接、按钮、表头、卡片标题、图表全用青色
- 没有区分主色 / 强调色 / 信息色 / 成功色 / 警告色 / 危险色
- 红色按钮 (重置) 和绿色按钮 (刷新) 放在一起没有视觉权重区分
- **修复**: 建立完整的语义色彩系统，控制青色使用范围

### 6. 排版层次不清
- 页面标题、卡片标题、表格标题大小差异不明显
- 字体只有一种，没有粗细变化
- 数字（如 31 总请求）应该用 Tabular Lining 数字字体
- **修复**: 建立 4 级标题层级 + 数字专用字体风格

### 7. 间距和对齐不一致
- 卡片之间间距不统一（有些 12px 有些 16px）
- 侧边栏导航项之间间距过大，整体松散
- 按钮组之间间距不一致
- **修复**: 建立 4px 基准网格间距系统 (4/8/12/16/24/32/48)

### 8. 表格样式粗糙
- 表头颜色太亮（青色），长时间看刺眼
- 行悬浮效果不明显
- 分页栏样式简陋，按钮太小
- **修复**: 表头用深色背景+浅色文字，行悬浮加微妙背景色变化

### 9. 表单控件原生感太重
- 日期选择器是原生 `<input type="datetime-local">`，跟深色主题完全不搭
- 下拉框是原生 `<select>`，跟自定义 UI 格格不入
- **修复**: 自定义表单组件样式，统一深色主题

### 10. 图表专业度不够
- 趋势图没有网格线，数据点不可交互
- 折线粗细单一，颜色对比不够
- X 轴标签显示 "05h 08h 11h" — 不够直观
- **修复**: 添加网格线 + 数据点标记 + 优化标签格式 (00:00, 06:00...)

- 行展开详情 — 好设计

## 迭代优先级建议

### Sprint 1（视觉基础 — 最紧急）
1. **图标系统**: 内联 SVG 图标替换所有方块占位符
2. **运行时间格式化**: 纳秒 → 人类可读
3. **StatCard 重新设计**: 响应式 + 图标 + 动效
4. **颜色系统规范化**: 建立语义色板
5. **排版层级**: 字号/字重/行高规范
6. **间距系统**: 4px 网格

### Sprint 2（交互打磨）
7. **空状态设计**: SVG 插画 + 引导文案
8. **表单控件风格统一**: 深色主题自定义样式
9. **表格打磨**: 表头/行悬浮/分页栏
10. **图表增强**: 网格线 + 数据点 + tooltip 优化

### Sprint 3（专业度提升）
11. **加载动画**: 骨架屏 / shimmer 效果
12. **过渡动画**: 页面切换 + 列表增删 + 模态框
13. **键盘导航**: Tab 顺序 + 回车确认 + ESC 关闭
14. **通知系统**: Toast 带图标 + 自动消失 + 堆叠
