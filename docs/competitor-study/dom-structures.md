## 关键设计模式总结

### 信息层级
- **NGSOC**: 概览(1个数字) → 趋势(折线图) → 分类(卡片) → 详情(表格)
- **天眼**: 整体视图(4卡片) → 分类统计(矩阵) → TOP排行(表格) → 详情(展开)
- **防火墙**: 状态行(8卡片) → 面板网格(6面板) → 表格/图表

### 颜色语义
- NGSOC: 低危绿/中危黄/高危橙/危急红
- 天眼: 蓝色主题, 紫色高亮卡片
- 防火墙: 绿色正常/黄色警告/红色危险, 五级风险色(红橙黄蓝绿)

### 交互模式
- 所有产品: 数字可点击跳转详情
- 天眼: 面板可配置/自定义布局/恢复默认
- 防火墙: 面板可关闭/折叠
- NGSOC/天眼: 自动刷新可控
